# Simple Tools MVP

Simple Tools Hub + Prompt Pad + テキスト整形 の静的MVPです。

## 起動
`index.html` をブラウザで開きます。

## 公開向け説明
Simple Toolsは、余計な機能を削ぎ落とした小さなツールシリーズです。

初回版では、AI用プロンプトを保存、検索、コピーできる「プロンプトパッド」と、貼り付けた文章を軽く整える「テキスト整形」を使えます。ログインなし、外部APIなし、ブラウザ内保存だけで動きます。

## Scope
- ハブ画面
- プロンプトパッド画面
- ツールの有効/無効
- プロンプトの追加、更新、検索、コピー、削除
- 登録済みタグの再選択
- テキストの空白整理、空行削除、箇条書き化
- テキスト出力のコピー
- ブラウザ `localStorage` 保存

## データ保存
保存した内容は、利用中のブラウザの `localStorage` に保存されます。外部サーバーへの送信、クラウド同期、アカウント機能はありません。ブラウザのデータを削除すると、保存した内容も削除されます。

## 含めないもの
- ログイン
- クラウド同期
- AI API連携
- 課金
- デスクトップアプリ化
